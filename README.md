# コラ娘<br/>
縦、横、斜めに同じ色の石を3つ以上並べると消えていくどこかで見たような落ち物パズルです。<br/>

カーソルキー ← →：横移動<br/>
カーソルキー ↓：高速落下、石が消える演出のスキップ<br/>
Zキー：石の順番入れ替え<br/>
